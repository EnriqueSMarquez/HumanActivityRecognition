# HumanActivityRecognition

Repository to train models on HAR datasets.
Datasets in ./data/:
-	Opportunity
-	PAMAP2
-	DaphNet Freeze Gait

Models in ./models/:
-	MLP
-	ResNet
-	Dilated ResNet

main.py trains ResNet and Dilated ResNet on the specified dataset
test_model.py tests given a run information and model parameters
